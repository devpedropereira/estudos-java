# EXERCÍCIO DE CONJUNTOS (HASHSET) 🚀

Esse código eu fiz para praticar a lógica de conjuntos em Java. O objetivo é ler alunos de três cursos (A, B e C) e contar quantos alunos existem no total, sem repetir quem está em mais de um curso ao mesmo tempo.

### 🧠 O que eu usei:
* **HashSet:** Usei essa coleção porque ela barra automaticamente qualquer ID repetido.
* **Scanner:** Para ler os dados direto do teclado.

### 🎯 Por que o HashSet?
Porque ele é muito rápido e me poupa de ter que ficar fazendo `if` para conferir se o aluno já tinha sido adicionado. O próprio Java faz esse trabalho sujo pra mim.
